# 红树河熊荣耀手表 5 蓝牙车钥匙原型

这是一个从零开始搭建的原型仓库，用来验证“荣耀手表 5 控制红树河熊蓝牙车钥匙”的核心链路是否可行。

当前版本包含：

- 可移植的车钥匙命令协议编码器/解码器
- 带挑战应答的会话状态机
- 可替换的 BLE 适配层接口
- 一个面向荣耀手表 5 真机对接的 BLE 适配器骨架
- 一个本地模拟设备，方便先做开发验证
- 一个面向手表交互流的控制器原型

## 快速开始

```bash
npm install
npm run dev
```

运行测试：

```bash
npm test
```

构建：

```bash
npm run build
```

## 目录结构

- `src/core`: 协议、状态机、通用类型
- `src/adapters`: BLE 适配器和模拟设备
- `src/watch`: 面向手表 UI 的控制器层
- `src/simulator`: 本地命令行入口
- `docs`: 荣耀手表 5 接入说明和后续落地建议

## 当前假设

由于荣耀手表 5 面向第三方应用的蓝牙车钥匙开放文档并不明确，这个仓库先把“协议核心”和“设备交互流”抽象成可迁移模块。后续接入真机时，主要替换的是 BLE 平台 API 层和 UI 壳层。
